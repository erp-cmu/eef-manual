\section*{คำนำ}
ในปัจจุบัน การบริหารโครงการวิจัยและการปฏิบัติการหนุนเสริมสนับสนุนวิชาการทุนนวัตกรรมสายอาชีพชั้นสูงมีความซับซ้อนมากขึ้น ทั้งในด้านการติดตามผลการดำเนินงาน การจัดเก็บข้อมูลเชิงโครงสร้าง และการจัดทำรายงานต่อหน่วยงานที่เกี่ยวข้อง ส่งผลให้การใช้เพียงเอกสารกระดาษหรือไฟล์อิเล็กทรอนิกส์ทั่วไปอาจไม่เพียงพอต่อการบริหารจัดการข้อมูลจำนวนมากอย่างเป็นระบบและต่อเนื่อง

ระบบ Enterprise Resource Planning (ERP) ที่พัฒนาบน Frappe Framework จึงถูกนำมาใช้เป็นเครื่องมือสำคัญในการบริหารจัดการข้อมูลโครงการ การบันทึกกิจกรรมการหนุนเสริม การติดตามรายบุคคล การประเมินสวัสดิภาพและการพัฒนาผู้รับทุน รวมถึงการจัดทำรายงานสรุปในมุมมองต่าง ๆ ที่รองรับทั้งผู้ปฏิบัติงานภาคสนาม ผู้ประสานงานโครงการ และผู้บริหารที่ต้องใช้ข้อมูลเพื่อการตัดสินใจ ระบบดังกล่าวช่วยลดความซ้ำซ้อนของงานเอกสาร เพิ่มความถูกต้องของข้อมูล และทำให้การสื่อสารระหว่างผู้มีส่วนเกี่ยวข้องเป็นไปอย่างมีประสิทธิภาพยิ่งขึ้น [web:18][web:19]

คู่มือฉบับนี้จัดทำขึ้นเพื่อเป็นแนวทางให้ผู้ใช้งานสามารถเรียนรู้และใช้งานระบบ ERP ดังกล่าวได้อย่างเป็นขั้นตอน ตั้งแต่การเข้าถึงระบบและการตั้งค่าบัญชีผู้ใช้ การบันทึกข้อมูลในเมนูสำคัญต่าง ๆ อาทิ การหนุนเสริมด้านที่ 1.1 และ 1.2 การพัฒนาหลักสูตรและกระบวนการเรียนการสอน การจัดการข้อมูลความเสี่ยง การติดตามรายบุคคล ตลอดจนการบันทึกกิจกรรมและการประชุม รวมถึงการดูรายงานสรุปและข้อมูลจากแบบสำรวจในรูปแบบต่าง ๆ โดยมีตัวอย่างหน้าจอประกอบเพื่อช่วยให้ผู้ใช้งานสามารถปฏิบัติตามได้อย่างถูกต้อง

ผู้จัดทำหวังว่าคู่มือการใช้งานระบบ Enterprise Resource Planning เพื่อการบริหารโครงการวิจัยและปฏิบัติการหนุนเสริมสนับสนุนวิชาการทุนนวัตกรรมสายอาชีพชั้นสูงฉบับนี้ จะช่วยให้ผู้ใช้งานทุกระดับสามารถใช้ระบบได้อย่างคล่องตัวและเกิดประสิทธิผลสูงสุดในการบริหารจัดการข้อมูลโครงการ ทั้งนี้ หากมีข้อเสนอแนะหรือพบข้อบกพร่องประการใด ผู้จัดทำขอน้อมรับไว้เพื่อปรับปรุงและพัฒนาคู่มือฉบับนี้ให้สมบูรณ์ยิ่งขึ้นต่อไป

\section{เกี่ยวกับระบบ}

ระบบสนับสนุนการบริหารโครงการวิจัยและปฏิบัติการหนุนเสริมสนับสนุนวิชาการทุนนวัตกรรมสายอาชีพชั้นสูงที่นำเสนอในคู่มือนี้ พัฒนาขึ้นบน \textbf{Frappe Framework} ซึ่งเป็นเฟรมเวิร์กสำหรับพัฒนาเว็บแอปพลิเคชันแบบครบวงจร (Full-Stack) ที่รองรับทั้งส่วนจัดการฐานข้อมูล ส่วนตรรกะทางธุรกิจ และส่วนติดต่อผู้ใช้ภายในแพลตฟอร์มเดียว ทำให้สามารถออกแบบ ปรับแต่ง และเชื่อมโยงกระบวนการทำงานต่าง ๆ ได้อย่างเป็นระบบ โดยไม่จำเป็นต้องพัฒนาโครงสร้างพื้นฐานใหม่ทั้งหมด

หัวใจสำคัญของ Frappe คือแนวคิดการจัดเก็บข้อมูลในรูปแบบเอกสาร (DocType) ซึ่งช่วยให้สามารถกำหนดโครงสร้างข้อมูล ฟิลด์ ความสัมพันธ์ และข้อบังคับของข้อมูลแต่ละประเภทได้อย่างชัดเจน เหมาะกับการจัดการข้อมูลโครงการ ข้อมูลการหนุนเสริม สวัสดิภาพความเป็นอยู่ของผู้รับทุน การพัฒนานักศึกษาด้านวิชาการและวิชาชีพ รวมถึงบันทึกกิจกรรมและการประชุมในรูปแบบที่เป็นมาตรฐานเดียวกันทั้งระบบ

ระบบยังรองรับการกำหนดสิทธิ์การเข้าถึงตามบทบาท (Role-Based Permissions) ทำให้สามารถแยกขอบเขตการใช้งานระหว่างผู้บันทึกข้อมูล ผู้ตรวจสอบ ผู้ดูแลระบบ และผู้บริหารได้อย่างเหมาะสม ลดความเสี่ยงจากการเข้าถึงข้อมูลที่ไม่เกี่ยวข้อง และช่วยให้การกำกับดูแลเอกสารและกระบวนการอนุมัติเป็นไปอย่างโปร่งใสและตรวจสอบได้

ในด้านการติดตามผลและการตัดสินใจ ระบบมีเครื่องมือสำหรับสร้างรายงาน แดชบอร์ด และการสรุปข้อมูลในมุมมองต่าง ๆ เช่น สรุปผลการสำรวจสวัสดิภาพผู้รับทุน การติดตามหัวข้อบันทึกที่มีกำหนดติดตาม หรือภาพรวมการดำเนินงานตามด้านต่าง ๆ ของโครงการ ผู้ใช้สามารถกรอง ดู และวิเคราะห์ข้อมูลได้แบบเรียลไทม์ ทำให้สามารถใช้ข้อมูลจากระบบในการวางแผนและปรับปรุงการดำเนินงานโครงการได้อย่างมีประสิทธิภาพ

ด้วยคุณลักษณะเหล่านี้ ระบบที่พัฒนาบน Frappe Framework จึงเหมาะสำหรับหน่วยงานที่ต้องการเครื่องมือดิจิทัลเพื่อบริหารจัดการข้อมูลโครงการอย่างเป็นระบบ รองรับทั้งปริมาณข้อมูลที่เพิ่มขึ้นในอนาคต และการปรับแต่งฟังก์ชันใหม่ ๆ ให้สอดคล้องกับรูปแบบการดำเนินงานและข้อกำหนดของทุนนวัตกรรมสายอาชีพชั้นสูงที่อาจเปลี่ยนแปลงไปตามช่วงเวลา

\section{การเข้าถึงแอปพลิเคชัน และการตั้งค่าบัญชีผู้ใช้}
บทนี้อธิบายขั้นตอนการเริ่มต้นใช้งานระบบ ตั้งแต่การเข้าใช้งานแอปพลิเคชันผ่านเว็บเบราว์เซอร์ การเข้าสู่ระบบด้วยชื่อผู้ใช้และรหัสผ่าน ไปจนถึงการตั้งค่าบัญชีผู้ใช้ให้เหมาะสมกับการทำงานของแต่ละคน โดยเนื้อหาจะครอบคลุมการเข้าถึงหน้าหลัก การเปิดเมนูบัญชีผู้ใช้ การแก้ไขข้อมูลส่วนตัว และการเปลี่ยนรหัสผ่าน รวมทั้งการบันทึกการเปลี่ยนแปลงให้มีผลในครั้งถัดไป เพื่อให้ผู้ใช้งานสามารถใช้งานระบบได้อย่างปลอดภัยและสอดคล้องกับบทบาทหน้าที่ของตนในกระบวนการบริหารโครงการวิจัยและการปฏิบัติการหนุนเสริมฯ
\begin{enumerate}
  \item เข้าไปที่เว็บไซต์ \url{https://eef.iecmu.com} ใส่ชื่อผู้ใช้และรหัสผ่านเพื่อเข้าสู่ระบบ \rimg{images/2026-03-17-06-24-49.png}{หน้าจอเข้าสู่ระบบของแอปพลิเคชัน}
  \item หลังจากเข้าสู่ระบบสำเร็จ คุณจะเห็นหน้าหลักของแอปพลิเคชัน ซึ่งมีเมนูต่าง ๆ ให้เลือกใช้งาน \rimg{images/2026-03-17-14-35-46.png}{หน้าหลักของแอปพลิเคชันหลังจากเข้าสู่ระบบ}
  \item คลิ้กที่เมนูด้านล่างซ้ายเพื่อดูเมนูบัญชีผู้ใช้และการตั้งค่า \rimg{images/2026-03-17-14-52-17.png}{เมนูบัญชีผู้ใช้และการตั้งค่า}
  \item เลือกเมนู ``User Settings" เพื่อปรับแต่งข้อมูลส่วนตัวและการตั้งค่าต่าง ๆ \rimg{images/2026-03-17-15-07-47.png}{เมนูการตั้งค่าผู้ใช้}
  \item เปลี่ยนข้อมูลส่วนตัวในหน้านี้ \rimg{images/2026-03-17-15-10-57.png}{หน้าเปลี่ยนแปลงข้อมูลส่วนตัว}
  \item เลือกที่แทป ``Settings" และเพื่อทำการเปลี่ยนแปลงรหัสผ่าน \rimg{images/2026-03-17-15-36-23.png}{แทปการตั้งค่าเพื่อเปลี่ยนรหัสผ่าน}
  \item กดปุ่ม ``Save" เพื่อบันทึกการเปลี่ยนแปลง \rimg{images/2026-03-17-15-32-32.png}{การบันทึกการเปลี่ยนแปลงข้อมูลส่วนตัว}
\end{enumerate}

\section{การบันทึกการหนุนเสริมด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน}
\label{sec:1.1}
บทนี้นำเสนอขั้นตอนการบันทึกข้อมูลการหนุนเสริมด้านที่ 1.1 ซึ่งมุ่งเน้นสวัสดิภาพและความเป็นอยู่ของผู้รับทุนในมิติต่าง ๆ อาทิ ข้อมูลพื้นฐานด้านการศึกษา สถาบัน หลักสูตร รุ่น รวมถึงรายละเอียดการหนุนเสริมในแต่ละครั้ง ทั้งนี้ การบันทึกข้อมูลอย่างเป็นระบบในส่วนนี้มีความสำคัญต่อการติดตามคุณภาพชีวิตผู้รับทุน การประเมินผลการดำเนินโครงการ และการจัดทำรายงานต่อหน่วยงานที่เกี่ยวข้อง เนื้อหาในบทนี้จะอธิบายขั้นตอนตั้งแต่การเลือกเมนูเข้าสู่หน้าบันทึก การกรอกข้อมูลในแต่ละแท็บ การบันทึกเอกสาร การแนบไฟล์ประกอบ ไปจนถึงการส่งเอกสารเข้าสู่กระบวนการตรวจสอบ เพื่อให้ผู้ใช้งานสามารถบันทึกข้อมูลได้อย่างถูกต้อง ครบถ้วน และสอดคล้องกับเกณฑ์ของโครงการฯ
\begin{enumerate}
  \item ในหน้าเมนูหลัก ให้เลือกเมนู ``ด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน" \rimg{images/2026-03-17-14-38-48.png}{หน้าจอเมนูด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน}
  \item เลือกที่ปุ่มสีดำด้านขวาบนเพื่อเข้าสู่หน้าบันทึกข้อมูล \rimg{images/2026-03-17-14-40-18.png}{หน้าจอการเลือกปุ่มเพื่อเข้าสู่หน้าบันทึกข้อมูล}
  \item ใส่ข้อมูลที่เกี่ยวข้องเช่น ข้อมูลการศึกษา สถาบัน หลักสูตร รุ่น หนุนเสริมครั้งที่ \rimg{images/2026-03-17-14-42-55.png}{หน้าจอการกรอกข้อมูลการหนุนเสริม}
  \item ใส่ข้อมูลอื่นๆโดยการเลือกแทปตต่างๆ \rimg{images/2026-03-17-14-44-48.png}{หน้าจอการเลือกแทปอื่นๆเพื่อกรอกข้อมูลเพิ่มเติม}
  \item เมื่อกรอกข้อมูลครบถ้วนแล้วให้กดปุ่ม ``Save" เพื่อบันทึกข้อมูลลงในระบบ \rimg{images/2026-03-17-14-46-29.png}{การบันทึกข้อมูลลงในระบบ}

\section{การสำรวจข้อมูลด้านที่ 2.1 การพัฒนาหลักสูตรและกระบวนการเรียนการสอน (อาจารย์ผู้รับผิดชอบหลักสูตร)}
หัวข้อนี้นำเสนอขั้นตอนการสำรวจข้อมูลด้านที่ 2.1 ที่เกี่ยวข้องกับการพัฒนาหลักสูตรและกระบวนการเรียนการสอน โดยมุ่งเน้นไปที่บทบาทของอาจารย์ผู้รับผิดชอบหลักสูตรซึ่งเป็นผู้ให้ข้อมูลหลัก ระบบจะใช้แบบสำรวจออนไลน์เป็นเครื่องมือรวบรวมความคิดเห็นและข้อมูลเชิงประจักษ์เกี่ยวกับการออกแบบหลักสูตร การจัดการเรียนการสอน และการพัฒนาผู้เรียนในมิติต่าง ๆ เพื่อสะท้อนภาพรวมของการดำเนินงานตามเป้าหมายของโครงการทุนนวัตกรรมสายอาชีพชั้นสูง

ในบทนี้ ผู้อ่านจะได้เรียนรู้ขั้นตอนการส่งลิงก์แบบสำรวจให้แก่อาจารย์ การตรวจสอบแบบฟอร์มที่ผู้ตอบแบบสำรวจมองเห็น การเข้าดูข้อมูลที่ส่งเข้าระบบ ตลอดจนวิธีการแก้ไขหรือลบข้อมูลที่เกี่ยวข้องในกรณีที่มีความคลาดเคลื่อน ข้อมูลที่ได้จากกระบวนการสำรวจนี้จะถูกนำไปใช้ประกอบการวิเคราะห์ ติดตามความก้าวหน้า และปรับปรุงหลักสูตรและกระบวนการเรียนการสอนให้สอดคล้องกับความต้องการของผู้เรียน สถานศึกษา และเป้าหมายของโครงการในระยะยาวต่อไป
\begin{enumerate}
  \item ส่งลิงก์แบบสำรวจให้กับอาจารย์เพื่อกรอกข้อมูล \url{https://cmu.to/eef-s21a}
  \item ตัวอย่างแบบสำรวจที่อาจารย์จะเห็นเมื่อเข้าลิงก์ \rimg{images/2026-03-18-15-45-25.png}{หน้าจอแบบสำรวจสำหรับอาจารย์}
  \item เมื่ออาจารย์กรอกข้อมูลครบถ้วนแล้ว สามารถเข้าดูรายละเอียดข้อมูลที่กรอกได้โดยการกดปุ่ม ``ด้านที่ 2.1 การพัฒนาหลักสูตรและกระบวนการเรียนการสอน (อาจารย์ผู้รับผิดชอบหลักสูตร)" \rimg{images/2026-03-18-15-49-14.png}{หน้าจอการเลือกปุ่มเพื่อดูรายละเอียดข้อมูลที่อาจารย์ได้กรอกไว้}
  \item คลิกที่ชื่อ อาจารย์ เพื่อดูรายละเอียด \rimg{images/2026-03-18-15-49-24.png}{หน้าจอการดูรายละเอียดข้อมูลอาจารย์}
  \item ผู้ดูแลระบบสามารถแก้ไขและลบข้อมูลที่อาจารย์ได้กรอกไว้ได้ \rimg{images/2026-03-18-15-49-42.png}{หน้าจอการแก้ไขและลบข้อมูลที่อาจารย์ได้กรอกไว้}
\end{enumerate}

\section{การดูข้อมูลสรุปด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน (นศ)}
บทนี้อธิบายการใช้งานหน้าจอสรุปข้อมูลด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุนจากมุมมองของนักศึกษา ซึ่งเป็นแหล่งข้อมูลสำคัญในการติดตามภาพรวมคุณภาพชีวิตและปัญหาที่ผู้รับทุนเผชิญในแต่ละช่วงเวลา ข้อมูลที่แสดงบนหน้าจอนี้ถูกประมวลผลจากแบบสำรวจที่นักศึกษากรอก และนำเสนอในรูปแบบตัวเลขสถิติและกราฟ เพื่อให้ผู้ดูแลโครงการสามารถมองเห็นแนวโน้ม จุดแข็ง และประเด็นที่ควรเฝ้าระวังได้อย่างรวดเร็ว ในหัวข้อนี้จะอธิบายวิธีการเข้าถึงหน้าสรุป การตั้งค่าฟิลเตอร์สำหรับเลือกดูข้อมูลตามสถาบันหรือรอบการหนุนเสริม ตลอดจนแนวทางการตีความผลสรุปเบื้องต้นเพื่อใช้ประกอบการวางแผนและปรับปรุงการหนุนเสริมให้ตอบโจทย์ความต้องการของผู้รับทุนมากยิ่งขึ้น
\begin{enumerate}
  \item ในหน้าเมนูหลัก ให้เลือกเมนู ``สรุปด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน (นศ)" \rimg{images/2026-03-18-15-51-57.png}{หน้าจอเมนูสรุปด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน (นศ)}
  \item ระบบจะแสดงข้อมูลสรุปในรูปแบบตัวเลขและบาร์กราฟ \rimg{images/2026-03-18-15-53-12.png}{หน้าจอการแสดงข้อมูลสรุปในรูปแบบตัวเลขและบาร์กราฟ}
  \item สามารถกำหนดฟิลเตอร์เพื่อดูข้อมูลสรุปตามสถาบัน และรอบการหนุนเสริมได้ \rimg{images/2026-03-18-15-54-11.png}{หน้าจอการกำหนดฟิลเตอร์เพื่อดูข้อมูลสรุป}

\section{การดูข้อมูลสรุปหัวข้อบันทึกที่มีกำหนดติดตาม}
บทนี้อธิบายวิธีการใช้งานเมนู “การดูข้อมูลสรุปหัวข้อบันทึกที่มีกำหนดติดตาม” ซึ่งเป็นเครื่องมือสำคัญสำหรับติดตามความคืบหน้าของประเด็นที่ถูกกำหนดให้มีการติดตามต่อเนื่อง โดยระบบจะแสดงข้อมูลสรุปในรูปแบบรายงานที่รวบรวมหัวข้อบันทึก วันที่นัดติดตามครั้งถัดไป และข้อมูลที่เกี่ยวข้องไว้ในที่เดียว ช่วยให้ผู้ใช้งานสามารถตรวจสอบได้อย่างรวดเร็วว่ามีหัวข้อใดที่ใกล้ถึงกำหนดติดตามหรือยังไม่ได้ดำเนินการ

ผู้ใช้งานยังสามารถใช้ตัวกรองข้อมูล (ฟิลเตอร์) เพื่อจำกัดขอบเขตการแสดงผล เช่น เลือกดูเฉพาะหัวข้อที่ต้องติดตามภายในช่วงวันที่หนึ่ง หรือเฉพาะหัวข้อที่เกี่ยวกับกลุ่มเป้าหมายบางประเภท ทำให้การบริหารจัดการการติดตามเป็นไปอย่างเป็นระบบ ลดโอกาสการหลงลืมหรือข้ามประเด็นสำคัญ และสนับสนุนการวางแผนการติดตามเชิงรุกของทีมโครงการโดยรวม
\begin{enumerate}
  \item ในหน้าเมนูหลัก ให้เลือกเมนู ``สรุปหัวข้อบันทึกที่มีกำหนดติดตาม" \rimg{images/2026-03-18-15-57-46.png}{หน้าจอเมนูสรุปหัวข้อบันทึกที่มีกำหนดติดตาม}
  \item ระบบจะแสดงข้อมูลสรุปในรูปแบบรายงาน \rimg{images/2026-03-18-15-58-11.png}{หน้าจอการแสดงข้อมูลสรุปในรูปแบบรายงาน}
  \item สามารถใช้การฟิลเตอร์เพื่อกรองข้อมูลตามคอลัมน์ต่างได้ เช่น หัวข้อที่จะติดตามครั้งต่อไป หรือวันที่ในการติดตามครั้งถัดไป \rimg{images/2026-03-18-15-59-53.png}{หน้าจอการใช้ฟิลเตอร์เพื่อกรองข้อมูลตามคอลัมน์ต่างๆ}
\end{enumerate}
