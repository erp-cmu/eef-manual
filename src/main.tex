\section*{คำนำ}

ปัจจุบันการบริหารและการจัดการโครงการมีความซับซ้อนมากขึ้น ทั้งในด้านการติดตามความก้าวหน้า การจัดเก็บข้อมูล และการรายงานผล ทำให้จำเป็นต้องมีเครื่องมือดิจิทัลที่ช่วยสนับสนุนการทำงานอย่างเป็นระบบและมีประสิทธิภาพ คู่มือเล่มนี้จัดทำขึ้นเพื่ออธิบายขั้นตอนและวิธีการใช้งานแอปพลิเคชันสนับสนุนการจัดการโครงการอย่างเป็นลำดับ เหมาะสำหรับผู้ใช้งานที่ต้องการเรียนรู้การใช้งานระบบตั้งแต่ระดับพื้นฐานไปจนถึงการใช้งานฟังก์ชันหลักในชีวิตการทำงานประจำวัน

แอปพลิเคชันดังกล่าวถูกพัฒนาบน Frappe Framework ซึ่งเป็นแพลตฟอร์มโอเพนซอร์สที่มีความยืดหยุ่น สามารถปรับแต่งให้สอดคล้องกับกระบวนการทำงานของหน่วยงานได้ คู่มือฉบับนี้จึงมุ่งเน้นนำเสนอขั้นตอนการใช้งานที่สำคัญ เช่น การเข้าถึงระบบและการตั้งค่าบัญชีผู้ใช้ การบันทึกข้อมูลในเมนูต่าง ๆ รวมถึงตัวอย่างหน้าจอเพื่อช่วยให้ผู้ใช้สามารถปฏิบัติตามได้อย่างสะดวกและถูกต้องลดความคลาดเคลื่อนในการใช้งานจริง

ผู้จัดทำหวังเป็นอย่างยิ่งว่าคู่มือประกอบการใช้งานแอปพลิเคชันสนับสนุนการจัดการโครงการฉบับนี้ จะเป็นประโยชน์ต่อผู้ใช้งานทุกท่านในการเรียนรู้และประยุกต์ใช้ระบบให้เกิดประสิทธิภาพสูงสุด ทั้งนี้ หากมีข้อเสนอแนะหรือพบข้อบกพร่องประการใด ผู้จัดทำขอน้อมรับไว้เพื่อปรับปรุงและพัฒนาคู่มือให้สมบูรณ์และเหมาะสมยิ่งขึ้นต่อไป

\begin{flushright}
  มีนาคม 2569\\
  ผู้จัดทำ

\section{เกี่ยวกับระบบ}

  \item หลังจากเข้าสู่ระบบสำเร็จ คุณจะเห็นหน้าหลักของแอปพลิเคชัน ซึ่งมีเมนูต่าง ๆ ให้เลือกใช้งาน \rimg{images/2026-03-17-14-35-46.png}{หน้าหลักของแอปพลิเคชันหลังจากเข้าสู่ระบบ}
  \item คลิ้กที่เมนูด้านล่างซ้ายเพื่อดูเมนูบัญชีผู้ใช้และการตั้งค่า \rimg{images/2026-03-17-14-52-17.png}{เมนูบัญชีผู้ใช้และการตั้งค่า}
  \item เลือกเมนู ``User Settings" เพื่อปรับแต่งข้อมูลส่วนตัวและการตั้งค่าต่าง ๆ \rimg{images/2026-03-17-15-07-47.png}{เมนูการตั้งค่าผู้ใช้}
  \item เปลี่ยนข้อมูลส่วนตัวในหน้านี้ \rimg{images/2026-03-17-15-10-57.png}{หน้าเปลี่ยนแปลงข้อมูลส่วนตัว}
  \item เลือกที่แทป ``Settings" และเพื่อทำการเปลี่ยนแปลงรหัสผ่าน \rimg{images/2026-03-17-15-36-23.png}{แทปการตั้งค่าเพื่อเปลี่ยนรหัสผ่าน}
  \item กดปุ่ม ``Save" เพื่อบันทึกการเปลี่ยนแปลง \rimg{images/2026-03-17-15-32-32.png}{การบันทึกการเปลี่ยนแปลงข้อมูลส่วนตัว}

\section{บันทึกการหนุนเสริมด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน}
ในส่วนนี้จะอธิบายขั้นตอนการบันทึกข้อมูลการหนุนเสริมด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน โดยมีขั้นตอนดังนี้:
\begin{enumerate}
  \item ในหน้าเมนูหลัก ให้เลือกเมนู ``ด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน" \rimg{images/2026-03-17-14-38-48.png}{หน้าจอเมนูด้านที่ 1.1 สวัสดิภาพความเป็นอยู่ของผู้รับทุน}
  \item เลือกที่ปุ่มสีดำด้านขวาบนเพื่อเข้าสู่หน้าบันทึกข้อมูล \rimg{images/2026-03-17-14-40-18.png}{การเลือกปุ่มเพื่อเข้าสู่หน้าบันทึกข้อมูล}
  \item ใส่ข้อมูลที่เกี่ยวข้องเช่น ข้อมูลการศึกษา สถาบัน หลักสูตร รุ่น หนุนเสริมครั้งที่ \rimg{images/2026-03-17-14-42-55.png}{หน้าจอการกรอกข้อมูลการหนุนเสริม}
  \item ใส่ข้อมูลอื่นๆโดยการเลือกแทปตต่างๆ \rimg{images/2026-03-17-14-44-48.png}{หน้าจอการเลือกแทปอื่นๆเพื่อกรอกข้อมูลเพิ่มเติม}
  \item เมื่อกรอกข้อมูลครบถ้วนแล้วให้กดปุ่ม ``Save" เพื่อบันทึกข้อมูลลงในระบบ \rimg{images/2026-03-17-14-46-29.png}{การบันทึกข้อมูลลงในระบบ}
  \item หลังจากกดปุ่ม ``Save" แล้ว เอกสารจะอยู่ในสถานะ ``Draft" ซึ่งหมายความว่ายังไม่ถูกส่งให้ผู้ตรวจสอบและสามารถแก้ไขข้อมูลได้ \rimg{images/2026-03-17-15-49-11.png}{สถานะของเอกสารหลังจากบันทึกข้อมูล}
  \item เอกสารในสถานะ ``Draft" จะสามารถอัพโหลดไฟล์แนบเพิ่มเติมได้โดยการเลือกแทป ``Attachments" และกดปุ่ม ``Add Attachment" \rimg{images/2026-03-17-15-52-46.png}{การอัพโหลดไฟล์แนบเพิ่มเติมในสถานะ Draft}
  \item เมื่อต้องการส่งเอกสารให้ผู้ตรวจสอบ ให้กดปุ่ม ``Submit" เพื่อเปลี่ยนสถานะเป็น ``Submitted" \rimg{images/2026-03-17-15-50-05.png}{การส่งเอกสารให้ผู้ตรวจสอบ}
\end{enumerate}
